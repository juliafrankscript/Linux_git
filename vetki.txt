для полной очистки индекса
git reset

для просмотра разницы между рабочим каталогом и посл коммитом
git diff HEAD

для создания новой ветки и одновременного переключения на нее
git checkout -b name

для сравнения веток между собой
git diff master ..feature

для удаления локальной ветки
git branch -d name

для слияния веток
git merge name

для перезаписи последнего коммита
commit --amend -m'исправленный комментарий'

конфликты,причина и порядок решения.
конфликт возникает,когда 2 человека изменяют одни и те же строки в файле или 
разработчик удаляет файл,который в это время изменяет другой разработчик.
Для решения проблемы
1.открываем файл (nano,vim)
2.изменяем вручную так,как нам нужно
3.сохраняем,выходим
4.добавляем файл в индекс git add filename
5.git commit -m 'Поправил строки...'
